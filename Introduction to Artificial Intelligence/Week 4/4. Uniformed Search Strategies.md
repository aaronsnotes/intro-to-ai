**Uniformed search** (or Blind Search) strategies have no additional information about states beyond the problem definition.

# Breadth-First Search (BFS)
- **Strategy:** expands the **shallowest** unexpanded node first.
- **Implementation:** uses a **FIFO** (First-In, First-Out) queue for the frontier.
- **Completeness:** yes (if $b$ is finite).
- **Optimality:** yes, *if* path cost is a non-decreasing function of depth (e.g., all step costs are equal).

## Python implementation (Graph Search version):
```python
from collections import deque

def breadth_first_search(problem):
	#create root node
	node = Node(state=problem.initial_state)
	
	#check if initial state is goal
	if problem.is_goal(node.state):
		return node
	
	#initialise FIFO frontier with root
	frontier = deque([node])
	
	#initialise explored set to avoid loops
	explored = set()
	
	while frontier:
		#POP shallowest node
		node = frontier.popleft()
		explored.add(node.state)
		
		#expand node
		for action in problem.actions(node.state):
			child = child_node(problem, node, action)
			
			if child.state not in explored and child not in frontier:
				#check if goal is found
				if problem.is_goal(child.state):
					return child
				frontier.append(child)
	
	return None #failure
```

# Other strategies
- **Uniform-cost search:** expands lowest path cost $g(n)$ first (use a `PriorityQueue`).
- **Depth-first search (DFS):** expands deepest nodes first (use a `stack` or recursion).
- **Depth-limited search:** DFS with a depth limit.
- **Iterative deepening DFS:** calls depth-limited search with increasing limits.
