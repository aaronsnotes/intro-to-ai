# What is a solution?
- **Definition:** a solution is an **action sequence** that leads from the initial state to a goal state.
- **Solution quality:** this is measured by the **path cost function.**
- **Optimal solution:** an optimal solution has the **lowest path cost** among all possible solutions.

The role of the AI agent is to determine the correct actions needed to find the best solution possible.
